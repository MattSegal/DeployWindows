===== Deployment Workflow =====

Run bootstrap script (bootstrap_deploy.ps1)

	* Set ExecutionPolicy to Bypass
	* Make C:\Deploy directory
	* Install Chocolately
	* Install .NET 4.5 via choco
	* Install Powershell and WMF 5 via choco
	* Install MS SQL Server 2014
	* Install PackageManager (for Install-Module)
	* Install DSC Packages
		- xWebAdministration
		- xSQLServer
	* Install utilities
		- git
		- python
	* Enable convenience features
		- Set windows explorer options
		- Install sublime text 3
		- TODO: Copy $profile
	* Pull deployment scripts from GitHub
	* Reboot box

Run Powershell DSC
	* Set up Windows Features
		- DotNet 3.5
		- DotNet 4.5
	* Set up IIS
		- Enable IIS
		- Enable ASP.NET 4.5
		- Add the app pool
		- todo: move files from deployment to website
		* Add the website

Install mysql
	- choco install mysql
	- choco install mysql.workbench
